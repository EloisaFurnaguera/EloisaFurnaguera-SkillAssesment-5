AJAX meets Star Wars

There is a great API called SWAPI -- the Star Wars API. It can receive GET requests and sends back information in JSON format. It does not require any kind of authentication or token to use.

We want to use data from SWAPI to display some information on our page. Specifically, we are interested in getting information about the planet Tatooine and displaying the planet's terrain, climate, and population.

To Do:

  • add an event listener on the button with the id 'getInfo' so that when it is clicked, an AJAX request is made to SWAPI.
  • the AJAX request should go to the following route via a GET request: 'https://swapi.co/api/planets/1/'
  • after the AJAX request has been made, the Star Wars API will send back results. To explore what the results look like, just go here in your browser: https://swapi.co/api/planets/1.
  • write a success function for the AJAX call that:
    • gets the terrain, climate, and population information from the results of the AJAX request (remember, the results are JSON, so you can grab information just like you would from a regular JavaScript object).
    • puts each of those pieces of information into the respective elements on the page.

You may use Javascript and jQuery to do this. Do not update any of the HTML provided.
